# mash

If you don't have Mash installed and set to your path, do the following:

```
$ wget https://github.com/marbl/Mash/releases/download/v2.2/mash-Linux64-v2.2.tar
$ tar -xvf mash-Linux64-v2.2.tar
$ rm -rf mash-Linux64-v2.2.tar
$ export PATH="mash-Linux64-v2.2/:$PATH"
```

or

```
$ nano .bashrc
```

scroll to the bottom and type

```
export PATH="path/to/mash-Linux64-v2.1/:$PATH"
```

Once you're done with your edits, click Ctrl X. You'll get a prompt asking if you want to 

```
Save Modified Buffer? (Answering "No" will DISCARD changes)
```
type Y, then hit Enter.
